## About
Official website for **La Motivacion**, a premium fitness lounge based in Dhuri, 
Punjab. Built as a freelance project to give the gym a professional 
digital presence.

## Pages
- **Home** – Hero, facilities overview, sister branch
- **Facilities** – Equipment, timings, spa and supplement shop
- **Membership** – Standard and Premium pricing plans
- **Trainers** – Certified trainer profiles and testimonials
- **Contact** – Inquiry form and location details

## Getting Started

```bash
# Clone the repo
git clone https://github.com/shahilseth/la-motivacion-website.git

# Navigate to project
cd la-motivacion-website

# Install dependencies
npm install

# Run locally
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
